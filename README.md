# virtual-assistant

# How to run?
### Step 01 :
Clone the repository

```bash
git clone https://github.com/MayukhBaruah19/virtual-assistant.git
```
### STEP 02 :
Create a conda environment after opening the repository : 
```bash
conda create -p venv python=3.12.9 -y
```
activate the envirnment
```bash
conda activate venv/
```
